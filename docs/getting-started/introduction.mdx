skypydb is an open source Python reactive database build for real-time data storage and retrieval.

The database is designed to be highly scalable, flexible, and easy to use. It provides a simple and intuitive API for interacting with data, and gives you access to a dashboard for real-time monitoring and managing data.

Want to dive right in? Check out the Quickstart example to run your first skypydb application.
